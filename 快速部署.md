# 在线投票系统 - 快速部署指南

## 🚀 5分钟快速启动

### 方式一：本地开发运行（推荐用于快速体验）

#### 1. 启动Redis
```bash
# Windows
redis-server --port 6379
```

#### 2. 启动gRPC服务（新终端）
```powershell
cd D:\CStudy\voting-app\vote-system
$env:GRPC_ADDR=":9090"
$env:REDIS_ADDR="127.0.0.1:6379"
go run ./cmd/grpcserver
```

#### 3. 启动HTTP服务（新终端）
```powershell
cd D:\CStudy\voting-app\vote-system
$env:HTTP_ADDR=":8080"
$env:GRPC_ADDR="127.0.0.1:9090"
go run ./cmd/httpserver
```

#### 4. 访问应用
打开浏览器访问：http://localhost:8080/

---

### 方式二：Docker容器化部署

#### 1. 构建镜像
```bash
cd D:\CStudy\voting-app\vote-system\build\package
./docker-build.sh
```

#### 2. 运行容器
```bash
# 启动所有服务
docker run -d --name redis -p 6379:6379 redis:7-alpine
docker run -d --name grpcserver -p 9090:9090 -p 9091:9091 \
  -e GRPC_ADDR=:9090 -e REDIS_ADDR=redis:6379 -e HEALTH_ADDR=:9091 \
  grpcserver:latest
docker run -d --name httpserver -p 8080:8080 -p 8081:8081 \
  -e HTTP_ADDR=:8080 -e GRPC_ADDR=127.0.0.1:9090 -e HEALTH_ADDR=:8081 \
  httpserver:latest
```

---

### 方式三：Kubernetes一键部署

#### 1. 准备环境
- 已安装Kubernetes集群
- 已安装kubectl

#### 2. 执行部署
```bash
cd D:\CStudy\voting-app\vote-system\deployments\k8s
./apply-all.sh
```

#### 3. 访问服务
```bash
# 端口转发
kubectl port-forward -n voting-app svc/httpserver-service 8080:8080

# 访问：http://localhost:8080/
```
